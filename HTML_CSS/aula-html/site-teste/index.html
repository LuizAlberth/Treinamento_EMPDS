O que são tags?

AS tags são literalmente caixas para colocar os elementos de forma mais correta e facilitar/diferenciar tanto para usuário quanto para a máquina

Quando se quer dar ênfase para alguma frase deve-se usar a tag Strong ou b

Para esse belo efeito de linhas usa-se a tag hr, já a tag br quebra linhas. Para itálico usamos i e para sublinhar usamos u

Uma boa forma de formatar dados como nome e e-mail, é usando o small que diminui o que fica abaixo (exemplo sob o Título) esse efeito riscado é a tag del
